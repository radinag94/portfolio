## Run Locally

Clone the project

```bash
  git clone https://github.com/radinag94/portfolio.git
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Features

- Responsive Design
- Video Logo
- Hamburger Menu for smaller screens
- Links to portfolio projects
